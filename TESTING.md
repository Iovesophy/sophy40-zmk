# 動作確認手順

## 1. 新しいファームウェアをダウンロード

GitHub Actionsから最新のビルド（row2col版）をダウンロードして書き込んでください。

https://github.com/Iovesophy/sophy40-zmk/actions

## 2. テスト方法

### A. キーを押してみる
普通にキーを押して反応するか確認

### B. もし反応しない場合

#### 簡易テスト（ピンセットやワイヤーで）
XIAOのピンを直接ショートさせて確認：

**左側で試す:**
1. P0.28とP1.15をショート → ESCが入力されるはず
2. P0.28とP1.14をショート → Qが入力されるはず
3. P0.29とP1.15をショート → CTRLが入力されるはず

**もしこれで反応する場合:**
- スイッチかダイオードの配線に問題がある

**これでも反応しない場合:**
- ピン番号が間違っている可能性

## 3. それでもダメなら

以下の情報を教えてください：
1. ピンセットテストで反応したか？
2. KiCadの回路図（スクリーンショット）
3. ダイオードの向き（黒い線がどちら側か）

## 4. col2row版も試す

もし反応しない場合、元の`col2row`に戻したバージョンも試せます。
