# ピン配線テスト手順

## テスト方法

1. **導通テスト**
   - テスターやピンセットで、Row0とCol0を直接ショートさせる
   - ESCキーが入力されるか確認

2. **各ピンの確認**
   
   左側で以下のピンをショートさせて、対応するキーが入力されるか：
   
   | Row | Col | 期待されるキー |
   |-----|-----|--------------|
   | P0.28 | P1.15 | ESC |
   | P0.28 | P1.14 | Q |
   | P0.28 | P1.13 | W |
   | P0.29 | P1.15 | CTRL |
   | P0.29 | P1.14 | A |

## もし反応しない場合

### A. ダイオード方向を変更
`config/boards/shields/sophy40/sophy40.dtsi` の
```
diode-direction = "col2row";
```
を
```
diode-direction = "row2col";
```
に変更

### B. ピン番号が間違っている
実際の基板配線と設定が一致していない可能性

### C. プルアップ/プルダウンの変更
```
GPIO_PULL_DOWN
```
を
```
GPIO_PULL_UP
```
に変更してみる
